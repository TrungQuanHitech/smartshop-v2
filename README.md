# SmartShop v2 - Giao diện Quản lý Bán hàng & Kho hàng Master Design

**SmartShop v2** là một ứng dụng quản lý bán hàng (POS) và kho hàng hiện đại, được xây dựng với công nghệ React, TypeScript và Vite. Dự án tập trung vào trải nghiệm người dùng cao cấp với giao diện "Master Design", hiệu ứng Glassmorphism và các tính năng nghiệp vụ mạnh mẽ.

## ✨ Tính năng nổi bật

### 🛒 Điểm bán hàng (POS)
- **Giao diện Master Design**: Bố cục 2 cột tối ưu, sidebar giỏ hàng hiện đại.
- **Tìm kiếm thông minh**: Hỗ trợ tìm kiếm nhanh sản phẩm (F2) với giao diện bắt mắt.
- **Quản lý Thuế & Chiết khấu**: Tích hợp trình chỉnh sửa VAT và chiết khấu trực tiếp trong giỏ hàng.
- **Thanh toán đa phương thức**: Hỗ trợ Tiền mặt, Chuyển khoản (VietQR động) và Ghi nợ.
- **In hóa đơn chuyên nghiệp**: Xem trước hóa đơn 80mm trước khi in, thiết kế tối ưu cho máy in nhiệt.

### 📦 Quản lý Kho hàng
- **Danh mục sản phẩm**: Quản lý chi tiết thông số kỹ thuật, hình ảnh và phân loại.
- **Expandable Rows**: Xem nhanh thông tin chi tiết sản phẩm ngay tại bảng danh sách.
- **In Barcode**: Tích hợp công cụ tạo và in tem mã vạch cho sản phẩm.

### 🕒 Lịch sử đơn hàng
- Theo dõi toàn bộ giao dịch đã thực hiện.
- Hỗ trợ xem chi tiết và in lại hóa đơn bất cứ lúc nào.

## 🚀 Công nghệ sử dụng
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion (Animations), Lucide React (Icons)
- **State Management**: Zustand
- **Tiện ích**: React Barcode, VietQR Integration, XLSX Export

## 🛠 Cài đặt và Chạy thử

1. **Cài đặt dependencies**:
   ```bash
   npm install
   ```

2. **Chạy ở chế độ phát triển**:
   ```bash
   npm run dev
   ```

3. **Xây dựng bản sản xuất**:
   ```bash
   npm run build
   ```
